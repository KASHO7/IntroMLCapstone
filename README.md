## Folder Descriptions

| Folder | Purpose |
|--------|--------|
| `/data` | Contains the dataset used for model training and testing. |
| `/models` | Five individual model notebooks + comparison notebook. |
| `/utils` | Helper functions for preprocessing and metric exporting. |
| `/metrics` | Output folder storing JSON files generated after model evaluation. |
